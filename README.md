## Memoria

Memoria is mobile application to hire a professional photographer to capture beautiful memories from a pre-graduation, graduation, wedding, and event.

## Tools

* Android Studio
* Java Language

## Dependencies

* Material
* Vector Drawable
* Navigation
* Google Play Service Auth
* Firebase Auth
* Auto Image Slider
* Glide
* Circle Image View
* Spin Kit
